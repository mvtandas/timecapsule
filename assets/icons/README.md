# Map Icons

## Capsule Marker Icon

The `capsule-marker.png` file is the icon used for all capsule markers on the map.

### Requirements:
- Transparent PNG format
- Recommended size: 120x120px (will scale to 40x40 on map)
- No background or border
- Should be visible on both light and dark map themes

### Usage:
Used in `DashboardScreen.tsx` for all map markers (public, private, locked).

### How to Replace:
1. Save your new icon as `capsule-marker.png` in this directory
2. Ensure it's a transparent PNG
3. Restart the app to see changes
